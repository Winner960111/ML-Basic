# 开始学习分类方法

## 地方性的话题：美味的亚洲与印度菜肴 🍜

无论是在亚洲亦或是在印度，饮食风俗在美味无比的同时，又在不同的地区各具特色。让我们来看一些地方美食的数据，并尝试理解一下他们的原料。

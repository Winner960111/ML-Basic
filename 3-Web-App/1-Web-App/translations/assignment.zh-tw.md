# 嘗試不同的模型

## 說明

現在，你已經能夠使用一個經過訓練的回歸模型來搭建web應用程序，那麽請你從前面的回歸課程中重新選擇一個模型來重做一遍web應用程序。你可以使用原來的風格或者其他不同的風格進行設計，來展示pumpkin數據。註意更改輸入以反映模型的訓練方法。


## 評判標準

| 標準                   | 優秀                                                 | 中規中矩                                                  |  仍需努力                      |
| -------------------------- | --------------------------------------------------------- | --------------------------------------------------------- | -------------------------------------- |
| | web應用程序按預期運行，並部署到雲端 | web應用程序存在缺陷或者顯示意想不到的結果 | web應用程序無法正常運行 |
